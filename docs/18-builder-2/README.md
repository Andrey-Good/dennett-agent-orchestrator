## Scope

This folder records deterministic proof that builder-authored drafts can use richer public Agent JSON contract surfaces without adding hidden builder-only state.

Stage 17 is accepted only for this bounded deterministic Builder 2.0 scope. The accepted scope is public-contract rich draft proof, builder repair pass, failure docs, schema/audit gates, and draft-only persistence.

It intentionally does not replace the Phase 17 owner folder. It exists because the Stage 17 proof task owns the exact `docs/18-builder-2/**` write scope.
